# Task-3
# 🚢 Titanic Dataset - SQL for Data Analysis

## 📌 Project Overview

This project demonstrates the use of **SQL** to analyze the Titanic dataset. Various SQL queries were written to explore passenger information, perform data analysis, generate insights, and demonstrate essential SQL concepts such as filtering, sorting, grouping, aggregate functions, subqueries, views, and indexing.

---

## 🎯 Objective

* Extract meaningful insights from the Titanic dataset using SQL.
* Practice writing analytical SQL queries.
* Demonstrate core SQL concepts used in data analysis.

---

## 🛠️ Tools Used

* **MySQL Workbench**
* **SQL**
* **Titanic Dataset (CSV)**

---

## 📂 Dataset

The dataset contains information about passengers aboard the RMS Titanic, including:

* Passenger ID
* Survival Status
* Passenger Class
* Name
* Sex
* Age
* Number of Siblings/Spouses
* Number of Parents/Children
* Ticket Number
* Fare
* Cabin
* Embarkation Port

---

### Data Retrieval

* SELECT
* WHERE
* ORDER BY

### Data Aggregation

* GROUP BY
* COUNT()
* SUM()
* AVG()
* MIN()
* MAX()

---

## 📊 Analysis Performed

The project includes SQL queries to:

* Display passenger information
* Filter passengers by gender and age
* Identify survivors and non-survivors
* Sort passengers by fare and age
* Count passengers by passenger class
* Analyze survival by gender
* Calculate average age and fare by passenger class
* Find highest and lowest ticket fares
* Count passengers by embarkation port
* Retrieve passengers paying above the average fare
* Identify the oldest passenger
* Create reusable SQL views
* Optimize query performance using indexes

---

## 📈 Key Insights

* Third Class had the highest number of passengers.
* Female passengers had a significantly higher survival rate than male passengers.
* First-class passengers generally paid higher fares and had better survival rates.
* Most passengers embarked from Southampton.
* Higher ticket fares were associated with higher survival probability.
* Passenger class, gender, and fare were among the strongest factors influencing survival.
